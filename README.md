# insurance-policy-system
A vehicle insurance management system developed in Python using CustomTkinter and SQLite. The application provides user authentication, insurance policy creation, vehicle data management, premium calculation, and policy storage in a local database through a user-friendly desktop interface

# Vehicle Insurance Policy System

Desktop application for vehicle insurance policy management and insurance premium calculation.

## Features

* User registration and authentication
* Vehicle information management
* Insurance premium calculation
* Policy generation and storage
* SQLite database integration
* Multi-step user interface

## Technologies

* Python
* Tkinter
* SQLite
* PIL (Pillow)
